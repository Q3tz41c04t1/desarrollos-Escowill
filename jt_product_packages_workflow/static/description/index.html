Product Packages Developement and Labels Printing

Generation of packages automatically and printing of packages labels

This module has been tested in enterprise version 12
Highlights Of The Application

- Automatic product packages generation when manufacturing order done
- Calculation of product packages to be generate based on product quantity in sale line and quantity in manufacturing order
- Product Packages Labels print report
- Separate Product Packages Labels print report
- Can preview separate labels

configuration of product, variants, packages and quantity etc. details










After configuration all information related product then go to sales and created sale order. In that configured products and package. You can also see how many total packages to be generate in sale line view







On sale confirming, there are number of manufacturing orders generated based on products. You can see sale order related detail in manufacturing order. You can see total packages to be come out after manufacturing orders







You can print total number of labels of packages using Label Printing report menu. The label is look like this






Added one more report Print Separately to print any separate label in the case of damage or any purpose. On click that menu you will see one wizard will open. Here you can select multiple label to be print. You can preview that labels by clicking Preivew button, you can also download that labels too.










On clicking MARK AS DONE in manufacturing order, it will create packages in delivery orders based on manufacturing quantity and package qauntity configured in product in sale line.
